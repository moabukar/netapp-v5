# NetApp V5

## Setup

```bash
cd backend
python3 -m venv venv
source venv/bin/activate   # On Windows, use `venv\Scripts\activate`
pip install -r requirements.txt
```

## Run

```bash
cd backend
python app.py
```

## Frontend

```bash
cd frontend
npm install
npm start
```

## All together

```bash
docker-compose up --build
```
